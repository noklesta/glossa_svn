Example 3: Brainjar.com "Revenge of the Menubar" Style Menu
To prove the flexibility of this menu system, the third example emulates yet another popular menu system, brainjar.com's "Revenge of the Menubar." The responsivness of this menu is very similar to the KDE style menu, except that now a "hovered" element is given a distinct style from an "active" element, which has an open submenu.
